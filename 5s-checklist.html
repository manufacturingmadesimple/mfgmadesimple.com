1
Baseline Measurement (Day 1)

Time Study: Measure how long operators spend searching for tools, parts, and information (track for 2 hours)
Photo Documentation: Take "before" photos of each work area from 4 angles
Inventory Count: Count total number of tools, parts, and supplies in work area
Safety Assessment: Document any safety hazards related to disorganization
üí° Pro Tip: Do this assessment when workers don't know they're being observed for the most accurate baseline data.

2
Red Tag Campaign

Prepare Red Tags: Create 50+ red tags with date, description, and decision fields
Set Up Red Tag Area: Designate holding area for questionable items
Apply Decision Criteria: Used daily = keep; weekly = nearby; monthly = storage; rarely = remove
Tag Everything Questionable: When in doubt, tag it‚Äîbetter safe than sorry
30-Day Review Process: Set calendar reminder to review red tag area in 30 days
‚ö†Ô∏è Common Mistake: Don't skip items because "we might need them someday." If you haven't used it in 6 months, red tag it.

üìç Set in Order (Seiton) - Days 3-4

3
Organize for Efficiency

Point-of-Use Storage: Place items as close as possible to where they're used
Frequency-Based Layout: Most-used items within arm's reach, less-used items further away
Visual Controls: Create shadow boards for tools, outline storage areas with tape
Clear Labeling: Label everything with large, easy-to-read text and/or pictures
Height Optimization: Place heavy items at waist level, light items higher/lower
üí° Pro Tip: Use the "30-second rule"‚Äîany item should be findable and returnable within 30 seconds by a new employee.

‚ú® Shine (Seiso) - Day 5

4
Clean to Inspect

Deep Clean Everything: Clean all equipment, surfaces, and storage areas thoroughly
Inspect While Cleaning: Look for wear, damage, leaks, or other abnormalities
Identify Root Causes: Find sources of dirt, debris, and contamination
Daily Cleaning Standards: Establish 5-minute daily cleaning routines for each area
Cleaning Supply Organization: Organize cleaning supplies for easy access

üìä Standardize (Seiketsu) - Day 6

5
Create Standards

Document Procedures: Write simple, visual procedures for Sort, Set in Order, and Shine
Create Audit Checklist: Develop scoring system for 5S compliance (1-5 scale)
Assign Responsibilities: Clearly define who does what, when, and how often
Post Visual Standards: Display before/after photos and standards at each workstation
Training Plan: Schedule training for all employees on new standards

üîÑ Sustain (Shitsuke) - Day 7 & Ongoing

6
Make it Stick

Daily Audits: Conduct brief 5-minute audits every morning for first 30 days
Weekly Team Reviews: Discuss successes, challenges, and improvements
Recognition System: Celebrate good 5S behavior and improvements
Continuous Improvement: Encourage suggestions for making 5S even better
Expand to Next Area: Once stable, implement 5S in another work area
